# 아이템 55. 옵셔널 반환은 신중히 하라
## 특정 조건에서 값을 반환할 수 없을때...
### 자바 8 이전
- `null` 반환
  - `null` 처리 코드를 추가해야하고, null을 저장한 이상 항상 `NullPointerException`이 발생할 가능성이 남아있게된다
- 예외 던지기
  - 예외 생성 시 스택 추적 전체를 캡처하는 비용이 만만치 않다
